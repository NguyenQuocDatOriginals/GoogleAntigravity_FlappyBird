## Welcome to Flappy Bird Repository

### You will find in this repo the following stuff

* A 2.5D Flappy Bird game built with Godot Engine 4.7 and GDScript
* The game allows users to control a bird by pressing Space or clicking the mouse to flap and avoid pipes
* The goal is to achieve the highest possible score by avoiding obstacles
* A Vietnamese countryside-themed experience with procedural 3D environments, smooth animations and real-time gameplay

### Technologies used

* Godot Engine 4.7 (Forward+) - Open-source game engine powering the 2.5D rendering
* GDScript - Godot's built-in scripting language for game logic and behavior
* Procedural 3D Generation - Environments, obstacles, and decorations created entirely through code
* Godot Physics (CharacterBody3D / StaticBody3D) - Handling bird movement and collision detection

### Features

* 2.5D gameplay combining 3D visuals with classic side-scrolling mechanics
* Vietnamese countryside theme with bamboo groves, village houses, and golden-hour sky
* Dynamic pipe generation with randomized gaps for an engaging experience
* Parallax scrolling background elements for depth and immersion
* Built-in sound manager for flap, score, hit, and background audio
* Vietnamese-localized UI optimized for 1280×720 landscape display
* Simple controls with spacebar or mouse click to flap
* Score tracking with best score persistence to challenge yourself
* MSAA anti-aliasing for smooth 3D visuals

### How to run

1. Download and install [Godot Engine 4.7](https://godotengine.org/download)
2. Clone this repository
3. Open the project in Godot Engine
4. Press **F5** or click the **Play** button to run the game
